БАЗОВАЯ КОНФИГУРАЦИЯ БЛЭКБОКСА ДЛЯ ПРОЕКТА ИНФОРМАТИКА-21

Данная конфигурация Блэкбокса придумана для облегчения сопровождения разных конфигураций Блэкбокса, используемых на разных уровнях обучения (полностью русифицированная школьная, частично русифицированные университетские и т.п.).

Архив blackbox15i21base доступен с сайта проекта Информатика-21, см.
http://www.inr.ac.ru/~info21/software.htm

СОСТАВ

Оригинальная инсталляция BlackBox Component Builder 1.5 
+ исправления и русификация из пакета di21 (почищенный пакет Дельта от oberoncore.ru; подробнее см. соотв. документ в папке i21sys), 
+ средства пакета i21sys
+ русскоязычная документация.

Никаких учебных материалов (кроме самой документации) в этом пакете нет.
Никаких удалений из оригинальной конфигурации не было сделано.

Все пакеты по отдельности, а также оригинальный дистрибутив Блэкбокса 1.5
доступны вместе с инструкциями по установке
с сайта проекта Информатика-21, 
см. http://www.inr.ac.ru/~info21/software.htm


УСТАНОВКА
Достаточно разархивировать архив в любую папку.


ЗАПУСК
Простейший способ: двойным кликом по BlackBox.exe.

Альтернативный способ: двойным кликом по StartBlackBox.vbs -- но тогда 
нужно обеспечить, чтобы путь, прописанный в файле как значение переменной prg, 
правильно указывал на нужную инсталляцию Блэкбокса.
И тогда можно будет скопировать vbs-файл в любую папку, 
и двойным кликом по файлу запускать Блэкбокс в т.наз. серверном режиме,
причем папка, в которой лежит кликаемый vbs-файл будет рабочей (вторичной), 
т.е. все вновь создаваемые файлы Блэкбокс будет складывать в ней 
как если бы это была основная папка Блэкбокса.

Инструкции по организации работы в компьютерном классе даны отдельно в учебных комплектах.
