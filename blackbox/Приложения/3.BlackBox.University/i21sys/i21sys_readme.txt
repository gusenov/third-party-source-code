СОСТАВ И УСТАНОВКА ПАКЕТА i21sys

Пакет содержит небольшой набор простых, но полезных средств, отсутствующих в стандартном комплекте Блэкбокса.
В частности, средства капитализации ключевых слов, а также поддержку русскоязычной документации.

Пакет собран для проекта Информатика-21 2008-06-05 (первая версия 2008-04-08) Ткачевым Ф.В. (info21@inr.ac.ru).

Архив i21sys, доступный с сайта проекта Информатика-21, содержит **только исходники**.

Архив содержит:
	исходники модулей подсистемы i21sys вместе с документацией;
	исходник модуля Config (System/Mod/Config.odc) с добавлением, обеспечивающим экспорт вьюшек (View) в формате 

BMP;
	документы меню с добавлениями соответствующих команд;
	данный документ.

По сравнению с оригинальными версиями, версии меню и модуля Config в данном пакете содержат **только добавления**.

ДОБАВЛЕНИЯ В МЕНЮ (в документах добавления/модификации выделены цветом):

Dev
	окошко Log всегда открывается в верхнем левом углу;
	отдельная команда для русскоязычной документации.
Text 
	полезные команды, в том числе капитализация;
	отдельная контекстная команда для русскоязычной документации.
System --
	команда сохранения вьюшки (View) как Bitmap;
	Attributes
		чуть изменены цвета Red, Green, Blue -- чтобы не резало глаза;
		добавлена комбинация атрибутов Comment;
		добавлены цвета Emphasis, Debug;
		добавлен размер 7.

Обзор средств пакета дан в модуле i21sys/Docu/ru/Sys-Map.odc.

ИНСТРУКЦИЯ
по включению пакета i21sys в инсталляцию Блэкбокса.

--0--
Предполагается, что BlackBox уже инсталлирован из файла SetupBlackBox15.exe (6.59MB), который можно скачать либо с сайта компании Oberon microsystems, Inc. (http://www.oberon.ch/zip/SetupBlackBox15.exe), либо, например, отсюда: 
http://www.inr.ac.ru/~blackbox/rsrc/SetupBlackBox15.exe.

Предполагается также, что BlackBox русифицирован с помощью пакета di21, доступного вместе с данным с сайта проекта Информатика-21.

--1--
Скопировать содержимое архива в папку Блэкбокса (можно во вторичную);
там должны появиться папка i21sys и документы i21sys_readme.txt и i21sys_readme.odc.
Запустить Блэкбокс, открыть документ i21sys_readme.odc,
дочитать до этого места и кликнуть по коммандеру (черный кружок с восклицательным знаком), при компиляции ответить OK на несколько сообщений о создании новых папок:

DevCompiler.CompileThis 
i21sysBitmaps i21sysDocuRu i21sysEdit i21sysIn i21sysListings i21sysWindows Config

--2--
Перезагрузить Блэкбокс, чтобы активировать новый Config, а также новые меню.
